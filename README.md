## 1. What is SMS-Activate?

SMS-Activate is a popular platform for **temporary phone numbers** and **SMS verification**. Users rent virtual numbers to receive OTP codes when registering on apps and websites.

Common supported platforms: Telegram, WhatsApp, Google, Facebook, Instagram, TikTok, Discord.

---

## 2. Why look for an alternative?

Users often search for an **SMS-Activate alternative** when they want:

- A modern, fast dashboard
- Competitive pay-as-you-go pricing
- Broader service catalog
- Reliable auto-refund on failed activations
- Crypto payments without KYC
- REST API for automation

## 5. Pricing

Both SMS-Activate and SmsApe use **pay-as-you-go** pricing. Costs depend on country, service, and availability.

SmsApe shows the exact price **before purchase**. Many activations start from **$0.04–$0.30**. No subscription.

---

## 6. API integration

SmsApe offers a public REST API:

```
POST /api/v1/activations          → buy number
GET  /api/v1/activations/:id      → poll for SMS
POST /api/v1/activations/:id/cancel → cancel + refund
```

Full documentation: [smsape.com/developers](https://smsape.com/developers)

Related repo: [giorgitchk/smsape](https://github.com/giorgitchk/smsape)

## 8. FAQ

### Is SmsApe a good SMS-Activate alternative?

Yes. SmsApe offers similar core functionality with a large in-app catalog (221 countries, 3,735 services), top services at [/services](https://smsape.com/services), auto-refund, and a REST API.

### Is /services the full service list?

No. It shows **top services** enabled in admin. The full catalog is in the dashboard after sign-up.

### Does SmsApe require KYC?

No identity verification is required.

### How fast is SMS delivery?

Usually under 30 seconds.

### Is there an API?

Yes — [smsape.com/developers](https://smsape.com/developers)

### What payments are accepted?

Cryptocurrency: BTC, USDT, ETH, LTC, TON, XMR, XRP, SOL, TRX, USDC, DOGE.
